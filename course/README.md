## Course Plan

- **Institution:** UFERSA - Pau dos Ferros Campus - DETEC;
- **Instructor:** Pedro Thiago Valerio de Souza;
- **Original repository:** <https://github.com/pedrothiag/intelligent-systems>.


### I. Objectives

- Introduce the concept of Artificial Intelligence;
- Present common AI approaches;
- Present elementary Machine Learning concepts;
- Understand regression concepts;
- Study the main existing neural network architectures;
- Understand problem solving using uninformed, heuristic, population-based, and stochastic search algorithms;
- Present AI methods based on trees.

### II. Syllabus

- Introduction to Artificial Intelligence;
- Data analysis and preprocessing;
- Machine Learning and training algorithms;
- Linear and logistic regression;
- k-Nearest Neighbors (k-NN);
- Support Vector Machines (SVM);
- Clustering algorithms;
- Introduction to Artificial Neural Networks;
- Perceptron and Adaline;
- Multilayer Perceptron (MLP);
- Convolutional Neural Networks (CNN);
- Decision trees;
- Naive Bayes classifier;
- Search methodologies (uninformed and informed search);
- Genetic algorithms;
- Simulated annealing;
- Deep Learning concepts;
- Natural Language Processing concepts;
- Applications in Artificial Intelligence.
